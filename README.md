# الداشبورد التجميعية لكل الشركات — 2026

داشبورد واحدة تفاعلية (ملف HTML واحد ذاتي الاحتواء) تعرض قوائم الدخل الشهرية لأربع شركات (زاوية ريال البركة، ريال البركة، عبد الرحمن، الاسواق) و20 فرعًا، مع فلترة حسب الفترة (سنة/ربع/شهر) والنطاق (كل الشركات/شركة واحدة/فرع واحد)، وصفحة ملاحظات وانحرافات تلقائية.

## النشر على GitHub Pages

1. أنشئ مستودع (Repository) جديد على GitHub، أو استخدم مستودعًا موجودًا.
2. ارفع محتوى هذا المجلد (`index.html` و`.nojekyll`) إلى جذر المستودع (أو إلى فرع `gh-pages` أو مجلد `/docs` حسب إعداداتك).
3. من إعدادات المستودع: **Settings → Pages** اختر المصدر (Source) المناسب:
   - **Branch**: `main` (أو الفرع الذي رفعت عليه الملفات) والمجلد `/root` (أو `/docs` إذا رفعت هناك).
4. احفظ، وانتظر دقيقة أو دقيقتين حتى يظهر الرابط بالشكل:
   `https://<اسم-المستخدم>.github.io/<اسم-المستودع>/`

## ملاحظات تقنية

- الملف يعتمد على مكتبتين خارجيتين تُحمَّلان من الإنترنت مباشرة عبر روابط CDN (Chart.js وFont Awesome) — لذا يجب أن يكون الجهاز الذي يفتح الصفحة متصلاً بالإنترنت، لكن لا حاجة لأي خادم (server) أو بناء (build) خاص؛ الملف يعمل كصفحة ثابتة تمامًا.
- ملف `.nojekyll` موجود لمنع معالجة GitHub Pages التلقائية (Jekyll) للملف، وهي ليست ضرورية هنا لكنها ممارسة قياسية آمنة.
- البيانات مضمّنة داخل الصفحة نفسها (JSON ثابت) وليست مسحوبة من قاعدة بيانات أو API خارجي؛ لتحديث الأرقام مستقبلًا يجب استبدال الملف بنسخة محدّثة بنفس الطريقة (وليس تعديل الأرقام يدويًا داخل الملف).
